🛒 תרחיש 1: השוואת מחירים

סימולציה של חברת השוואת מחירים שסורקת אתרי מסחר אלקטרוני

🎯 מטרה
סריקת מחירי מוצרים מ-10+ אתרי קניות
אתרים לדוגמה:
• Amazon.com
• eBay.com
• Walmart.com
• Target.com
⚙️ פרמטרים
בקשות: 1,000-10,000/שעה
מדינות: US, UK, DE, FR
סשן: דביק (30 דקות)
פרוטוקול: HTTP/HTTPS
📊 מדדי הצלחה
• אחוז הצלחה: >95%
• השהיה ממוצעת: <3 שניות
• אחוז חסימה: <5%
🔍 מה לבדוק
• טירגוט גאוגרפי עובד?
• סשנים נשמרים?
• רוטציה מתפקדת?
• Headers נכונים?

🔍 תרחיש 2: סריקת תוצאות חיפוש

סימולציה של כלי SEO שסורק תוצאות Google

🎯 מטרה
סריקת דירוגים ותוצאות חיפוש
מנועי חיפוש:
• Google.com
• Bing.com
• DuckDuckGo
⚙️ פרמטרים
בקשות: 500-5,000/שעה
מדינות: 10+ גאוגרפיות
סשן: מתחלף
User-Agent: פרופילי Chrome
📊 מדדי הצלחה
• אחוז הצלחה: >90%
• אחוז CAPTCHA: <10%
• השהיה: <5 שניות
🔍 מה לבדוק
• רוטציית IP עובדת?
• טירגוט גאו מדויק?
• Browser fingerprint?

🔬 תרחישי בדיקה מתקדמים

🔗 בדיקות סשן דביק
• סשן דביק תחת עומס (100+ מקבילים)
• התנהגות פקיעת סשן
• העברת סשן כשעמית מתנתק
• רוטציה לפי דרישה באמצע סשן
📦 בדיקות רוחב פס
• הורדת קבצים גדולים (10MB, 100MB)
• העלאה דרך פרוקסי
• דיוק מדידת רוחב פס
• תגובות Streaming/Chunked
🔌 בדיקות פרוטוקול
• מנהרת HTTPS CONNECT
• תמיכת SOCKS5
• HTTP/2 (אם נתמך)
• WebSocket דרך פרוקסי
💥 בדיקות כשל והתאוששות
• ניתוק עמית באמצע בקשה
• טיפול ב-timeout של יעד
• דגרדציה מבוקרת תחת עומס
• התנהגות ניסיון חוזר אוטומטי
🛒 תרחיש אמיתי: זרימת מסחר עדיפות
סימולציית סריקת מחירים:
1. עמוד חיפוש → קבלת רשימות
2. עמוד רשימה → חילוץ מוצרים
3. עמוד מוצר → קבלת מחיר/פרטים
בודק רציפות סשן מלאה
📱 תרחיש אמיתי: רשתות חברתיות
סימולציית דפוסי API:
• טיפול ביעדים מוגבלי קצב
• שמירת טוקן אימות
• עימוד דרך תוצאות
• עקביות טביעת אצבע מובייל

# ניווט לתיקיית בדיקות
cd /root/clawd-secure/iploop-platform/tests/pilot

# בדיקה מהירה
./run-tests.sh quick

# חבילה מלאה (כל 15 הבדיקות)
./run-tests.sh full

# רשימת תיוג לפרודקשן
./run-tests.sh production

# בדיקות כבדות בודדות
./run-tests.sh security     # ביקורת אבטחה
./run-tests.sh concurrency  # מקרי קצה
./run-tests.sh connection   # שבירת connection pool
./run-tests.sh stability    # 10 דק׳ יציבות

🎯 מה חבילת הבדיקות תופסת

🔴 דליפות IP

מזהה כש-IP של עמית חשוף במקום IP של פרוקסי

🔴 כשלי סשן דביק

כשסשנים דביקים נשברים או מתחלפים באופן בלתי צפוי

🔴 פספוסי טירגוט גאו

מדינה/עיר שגויה מוחזרת לבקשות גאוגרפיות

🔴 שיעורי Timeout/שגיאות

השהיה גבוהה, כשלי חיבור, קפיצות שגיאות

🔴 צווארי בקבוק ברוחב פס

הורדות איטיות, throttling, בעיות streaming

🔴 כשלי התאוששות

מערכת לא מתאוששת כראוי אחרי שגיאות
